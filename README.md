## 📖 About

**Bookr** is a fully client-side appointment booking app delivered as a *single HTML file*.  
Complete booking flow: **Service → Provider → Date → Time Slot → Confirmation**  
Ideal for freelancers, clinics, consultants, and coaches.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🗂️ Service Selection | 4 services with duration & pricing |
| 👥 Staff Picker | Choose a provider with independent slot availability |
| 📅 Interactive Calendar | Past/weekend blocked, today highlighted, dots for available days |
| 🕐 Time Slot Picker | AM/PM slots; booked slots shown as strikethrough |
| 📋 Booking Summary Bar | Animated sticky panel before checkout |
| ✅ Confirmation Modal | Contact form + animated success screen |
| 📌 Appointments Tab | View and cancel upcoming bookings |
| 📱 Fully Responsive | Works on desktop, tablet, and mobile |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/faheem1309/sample-booking-app-.git

# Navigate into it
cd sample-booking-app-

# Open in browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html   # Linux
```

Or just **[view it live →](https://sample-booking-app-gray.vercel.app/)**
